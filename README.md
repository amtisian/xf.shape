**Usage**

Namespace:

    xmlns:utmdev="clr-namespace:Utmdev.Xf.Components;assembly=Utmdev.Xf.Switch"

Component

    <utmdev:Switch x:Name="switcher"
                                VerticalOptions="CenterAndExpand"
                                HorizontalOptions="CenterAndExpand"
                                IsOn="False"
                                HeightRequest="50"
                                WidthRequest="125"
                                CircleColor="White"
                                BackgroundColor="#2FC061" />
